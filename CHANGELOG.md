## [0.1.0] - Unreleased

This will be the first official release of Time Tracker.

### Added
- Complete time tracking application
- Frontend: Angular 19 with Material Design
- Backend: Express.js with TypeScript and MongoDB
- Features:
  - User authentication (JWT-based)
  - Customer management
  - Project management with tasks
  - Contract management with billing rates
  - Time entry tracking (manual and timer-based)
  - Report generation
  - Azure DevOps integration for importing work items
  - Dark/light theme support
